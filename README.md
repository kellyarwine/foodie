#Foodie

##How to run
* Install nodejs

* Install Grunt  
`npm install grunt -g`  
`npm install grunt-cli -g`

* Install Bower  
`npm install bower -g`

* Install Bower Dependencies  
`bower install`

* Install Node Dependencies  
`npm install`

* Compile the Coffeescript  
`grunt build`

* Run the web server at `http://localhost:8000`
`grunt web-server`
